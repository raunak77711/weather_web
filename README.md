# weather app 🌤️

a minimal, real-time weather app built with vanilla html, css & javascript.

![Preview](preview.png)

## features
- 🔍 search any city worldwide
- 🌡️ temperature, feels like, humidity, wind, pressure & visibility
- 🎨 vercel-inspired dark ui with smooth animations
- 📱 fully responsive

## tech stack
- html / css / javascript
- openweather api
